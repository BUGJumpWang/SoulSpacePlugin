# SoulSpacePlugin - A Minecraft Server Plugin
depends: Vault  
This plugin gives you a very big backpack, which can be upgraded.
## How to Use:
Type "/getbags" can spend the initial amount to get a feather named "[SSP] 灵魂空间", right-click to open.
